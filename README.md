# TowerWatch: Real-time Threat Detection System

TowerWatch is a Django-based web application designed for real-time threat monitoring and alerting. The system provides a comprehensive dashboard for security monitoring, alert management, and threat analysis.

## Features

- **Real-time Threat Detection**: Monitor and detect potential security threats in real-time
- **Interactive Dashboard**: User-friendly interface for monitoring system status and alerts
- **Alert Management**: Comprehensive alert system with severity levels and notifications
- **Threat Analysis**: Built-in threat engine for analyzing and categorizing security events
- **Admin Interface**: Easy management of alerts and system settings
- **User Authentication**: Secure login and role-based access control
- **Responsive Design**: Works on desktop and mobile devices
